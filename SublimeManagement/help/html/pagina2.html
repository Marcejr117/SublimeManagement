  • Albaran

Desde esta ventana podemos crear una lista de los albaranes de nuestros proveedores para tener contabilizados los productos, vamos a definir cada uno de los apartados de esta ventana

  1. Aqui se especifica el producto que vamos a incorporar a nuestor albaran, este debe seleccionarse mediante el cuadro que se abre al hacer clic en "...", el texto que aparece en el campo sera la referencia del producto seleccionado
  2. Especificamos el precio al que hemos comprado el producto
  3. la cantidad del producto que hemos comprado
  4. agregamos el producto con su precio y cantidad al albran
  5. nos limpia los campos que hemos relleganodo anteriormente
  6. columna de la tabla donde se especifica la referencia de producto
  7. columna de la tabla que espeficica la cantidad de este producto
  8. columna de la tabla que espeficica la el precio unitario de este producto
  9. nos muestra el importe total del albaran
  10. nos permite eliminar un registro del albaran en caso de habernos equivocado
  11. campo donde podremos la referencia que nos ayudara a identificar el albaran
  12. Boton que nos crea el albaran y lo almacena en la base de datos
  13. Tabla donde se muestran todos los albaren de la base de datos
  14. Tabla que muestra los datos del albaran seleccionado anteriormente
  15. boton que nos permite cargar todos los albaranes en caso de  que no esten cargados correctamente
  16. nos pemite eliminar un albaran de la tabla numero 13

  • Ventas

  1. Despliega un menu desde el cual seleccionar el producto a vender
  2. Especificar la cantidad de producto que vamos a vender
  3. Precio base del producto
  4. Descunto que le aplicamos
  5. Precio final del producto
  6. Permite especificar un descuento preestablecido
  7. Agrañade el producto al carrito de venta
  8. Devuelve a 0 todos los valores de los campos anteriores
  9. Tabla que muestra el total de producto que vamos a vender
  10. Permite veder el producto a un cliente de nuestra lista
  11. Precio final (suma de las cantidad por el precio de unidad)
  12. Despliega un menu desde el cual especificar el cliente
  13. Elimina el producto previamente seleccionado
  14. Se termina la venta y se agrega al panel 15
  15. Muestra el total de ventas y en caso de hacer clic en una, esta se detalla en el panel 16
  16. Tabla que muesta con detalle un albaran
  17. Muetra las ventas en el panel 15
  18. Elimina una venta seleccinada
  19. Imprime un ticket de la venta seleccionada

  • Clientes

Esta venta nos permite registrar los datos de nuestros Clientes, para que posteriormente puedas trabajar con sus datos en el apartado facturas

Vamos a explicar cada parte de esta ventana con mas detenimiento

  1. Campo para especificar el nombre de nuestro cliente
  2. Campo para especificar los apellidos de nuestro cliente
  3. Campo para especificar el DNI de nuestro cliente
  4. Campo para especificar el Telefono de nuestro cliente
  5. Campo para especificar el Correo Electronico de nuestro cliente
  6. Campo para especificar la Localidad de nuestro cliente
  7. Campo para especificar la direccion de nuestro cliente
  8. Campo para especificar si nuestro cliente es un proveedor
  9. Boton que permite crear el cliente "Ningun campo es requerido para crear un cliente"
  10. Boton que permite buscar un cliente, para filtrar nuestros cliente debemos rellenar los campos especificados anteriormente, ya sea con el nombre completo con una parte del, la busqueda funciona en todos los campos, es decir, en caso de que nos acordemos de ciertas partes del clientes
  11. Tabla que muestra todos los datos de los clientes creados
  12. Boton para cargar todos los clientes de la base de datos "en caso de que no lo haga automatico"
  13. Boton que permite eliminar un Cliente de la base de datos
